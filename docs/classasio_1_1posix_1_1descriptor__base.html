Detailed Description

The descriptor_base class is used as a base for the descriptor class as a place to define the associated IO control commands.

Member Typedef Documentation

◆ bytes_readable

typedef implementation_defined asio::posix::descriptor_base::bytes_readable

IO control command to get the amount of data that can be read without blocking.

Implements the FIONREAD IO control command.

Example
asio::posix::stream_descriptor descriptor(io_context);
...
asio::descriptor_base::bytes_readable command(true);
descriptor.io_control(command);
std::size_t bytes_readable = command.get();
